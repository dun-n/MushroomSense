Configure Home Assistant

The following is a basic configuration that can be added to Home Assistants configuration.yaml file. It will add an entity to Home Assistant for the CO2, Temperature, and Humidity measured by your Mushroom Sense sensor. See Home Assistant's website for more information on yaml

sensor:
  - platform: rest
    resource: 
    name: "Mushroom Sense Temperature"
    value_template: '{{value_json.temperature_C}}'
    unit_of_measurement: "Â°C"
    scan_interval: 10
  - platform: rest
    resource: 
    name: "Mushroom Sense CO2"
    value_template: '{{value_json.co2}}'
    unit_of_measurement: "ppm"
    scan_interval: 10
  - platform: rest
    resource: 
    device_class: "humidity"
    name: "Mushroom Sense Humidity"
    value_template: '{{value_json.relative_humidity}}'
    unit_of_measurement: "%"
    scan_interval: 10
